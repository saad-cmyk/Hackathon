VeriSight AI

Description
VeriSight AI is a project designed to detect deepfake images and videos and verify the authenticity and origin of digital media. It helps identify whether content is genuine, manipulated, or altered after creation, supporting trust and reliability in digital information.

Features
- Deepfake image and video detection  
- Digital provenance and content integrity verification  
- Clear authenticity results  

#Tech Stack
- Python  
- Deep Learning  
- OpenCV  
- TensorFlow / PyTorch  
- Flask / FastAPI  

Use Cases
- Media verification  
- Fake news detection  
- Digital forensics  
- Content authenticity checking  

Workflow
1. Upload an image or video  
2. Analyze media for manipulation  
3. Verify provenance and integrity  
4. Generate final authenticity result  

Goal
To reduce the spread of manipulated media and improve trust in digital content.
